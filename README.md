# PayCompute Pro

Built by Ananth

## SF Payroll Deductions Upload Automation

Automated tool to extract payroll deductions from Excel files and generate SF-compatible CSV uploads.

### Features
- Smart data validation
- 10 quality control checks
- Generate SF-compatible CSV
- No installation needed
- Works in any browser

### How to Use
1. Upload payroll Excel file
2. Select month and year
3. Enter sheet name
4. Click "Generate SF Upload"
5. Download CSV

### Requirements
- Excel file with deduction codes in Row 1
- Staff IDs in Column A
- Wage-Type-Mapping.xlsx in same folder
